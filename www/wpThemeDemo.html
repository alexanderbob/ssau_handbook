jQuery Mobile theme for Windows Phone 8

Welcome

This theme provides a Windows Phone user interface for Apache Cordova apps using jQuery Mobile on Windows Phone 8.

While the theme itself adds all wndows phone specific colors and fonts by default, please don't forget to follow user experience design guidelines.

To view DateTime picker and Theme Switcher plugins of the jQM theme for Windows Phone, please view from within an Apache Cordova app in Windows Phone 8.
